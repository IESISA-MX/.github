# IESISA

**IESISA** es una empresa mexicana especializada en **consultoría tecnológica**, **desarrollo de software a medida** y **acompañamiento en procesos de calidad**, incluyendo **ISO 27001**.

Actuamos como **socios tecnológicos** para organizaciones que requieren soluciones confiables, escalables y alineadas a estándares de calidad. 🤝

---

## Qué hacemos

### 🧠 Consultoría tecnológica
- Diagnóstico y mejora de arquitecturas de software  
- Acompañamiento técnico para equipos internos  
- Definición de buenas prácticas, procesos y flujos de trabajo  

### 💻 Desarrollo de software
- Sistemas web y móviles a medida  
- APIs, backends y plataformas internas  
- Integraciones con servicios de terceros y sistemas legacy  

### 📋 Seguridad de la información (ISO 27001)
- Apoyo en la implementación y mantenimiento de ISO 27001  
- Documentación técnica y de procesos  
- Alineación entre tecnología y sistemas de gestión de calidad
